# Slots-de-invetario-costumizado

Sistema de hotbar customizada para Roblox que substitui o CoreGui padrão por uma interface moderna e funcional.

## Funcionalidades

### Organização Persistente
Itens salvam a posição no slot por nome; não embaralham ao resetar.

### Drag-and-Drop
Arraste itens entre os slots 1 e 0 para organizar em tempo real.

### Animações
Transição de entrada em cascata e descida automática ao morrer.

### Foco em Combate
Ideal para jogos de tiro (FPS/TPS) pela troca rápida e visual limpo.

## Observações Importantes

### Sem Menu Expandido
Não há interface de grade para ver itens extras (o aspas não abre menu).

### Limite de Slots
Suporta até 10 itens visíveis diretamente na barra.

### Instalação
Colocar o código em um `LocalScript` dentro de `StarterPlayerScripts`.

## Controles

### Teclas 1–9 e 0
Equipar / Desequipar.

### Clique e Arraste
Reorganizar slots.
